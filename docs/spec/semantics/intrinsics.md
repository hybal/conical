# Intrinsic Semantics

This document specifies the usage and definition of compiler intrinsics, as well as a list of defined intrinsics with their signature.

## Intrinsic Definition

An intrinsic is a function that is defined in the compiler itself and is replaced by code generated by the compiler.
They are used for platform-specific code, to define functionality that operators usually take, and to communicate things to the compiler itself.
